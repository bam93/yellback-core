# Attributions

Per-clip license record for audio bundled with `yellback-core`. Every clip
committed under `Resources/Packs/` must appear here with its source and
license. Contributions that add or replace audio must update this file in the
same commit.

## Accepted licenses

- **CC0 1.0** — public domain dedication, no attribution required
- **CC-BY 4.0** — attribution required (record the attribution line here)
- **CC-BY-SA 4.0** — attribution + ShareAlike (record both here)

All other licenses (CC-BY-NC, proprietary, unclear provenance) are rejected.
